================================================================================
                           NEW FILES CREATED (3)
================================================================================

1. DesktopPet/AudioManager.swift
   - Wraps faster-whisper (STT) + Kokoro (TTS)
   - Streams microphone → whisper server
   - Posts dialogue → Kokoro, with fallback to SystemTTSFallback
   - No cloud APIs, all local

2. DesktopPet/DialogueNaturalness.swift
   - Enhances raw dialogue with emotional pauses
   - Happy = quick, punchy responses
   - Sad/sleepy = slow with ellipses
   - Breaks long sentences at ~7 words for breathing points
   - Prevents repetition (tracks 20-line history, rejects >60% overlap)

3. DesktopPet/SystemTTSFallback.swift
   - macOS native TTS (AVSpeechSynthesizer)
   - Used if Kokoro server unavailable
   - Maps emotions → pitch/speed (happy=faster+higher, sad=slower+lower)
   - Zero setup required, immediate fallback

================================================================================
                           MODIFIED FILES (3)
================================================================================

1. DesktopPet/VoiceInputManager.swift
   - REWRITTEN: Removed Apple SFSpeechRecognizer (cloud API)
   - Now wraps AudioManager for local whisper-based transcription
   - Added speak(text, emotion) method with emotion-aware TTS
   - Maintains backward-compatible API

2. DesktopPet/AIEngine.swift
   - Changed default provider: Gemini → LocalOllamaProvider (Gemma 2B)
   - Added dialogue history tracking (prevent repetition)
   - Calls DialogueNaturalness.enhanceForSpeech() on all outputs
   - Added emotional tone hints to system prompts (bouncy/wistful/clipped)

3. DesktopPet/PetScene.swift
   - Removed AVSpeechSynthesizerDelegate protocol
   - Removed local AVSpeechSynthesizer instance
   - Updated say() method to use VoiceInputManager.speak()
   - Cleaned up showListeningState() and showDictationState()
   - All TTS now emotion-aware with natural pauses
